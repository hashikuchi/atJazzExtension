@Jazz Message Extension の使い方
このファイルはChrome拡張機能です。
Chromeをお持ちでない方は以下のリンクからダウンロードしてください。
http://www.google.co.jp/intl/ja/chrome/browser/

インストール方法
1. Chromeのメニューから 「ウィンドウ」->「拡張機能」を選択します。
2. 「パッケージ化されていない拡張機能を読み込む」をクリックし、「atjazzExtension」フォルダを選択します。
3. 画面に「@Jazz Message Extension」と表示が出ればインストール完了です。 

使用方法
1.マイページの「メッセージ」->「新規メッセージ」->「メッセージを送る」からメッセージを作成します。宛先は”誰でもOK”です。
2.メッセージ作成画面で「送信する」を選択して確認画面に進むと、「全員に送信する」というボタンがありますのでそちらをクリックすると、全員にメッセージが送られます。
「送信完了しました！」とメッセージが出れば送信完了です。特に画面遷移などしませんが、正常に処理されています。

種明かしとか注意点とか
- この機能は同じメッセージを宛先を変えて人数分送っているだけですので、実行後は送信トレイにすごい数のメッセージがたまります。びっくりしないでください。
- 宛先リストは atjazzExtensionフォルダ内、messageExtension.jsのIDListにべた書きで実装されています。怒らないでください。
- 宛先リストを変更したいときは、上記ファイル内IDListにIDを追加または削除してください。
　IDは、各メンバー情報のページURL末尾「?memid=XXXXX」の「XXXXX」の部分です。
